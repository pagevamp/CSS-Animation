  • Transition

    CSS transition is a property that lets you create gradual transitions between the values of CSS properties. The behavior of these transitions can be controlled by specifying their timing function, duration, delay.

    Properties

    • transition
    • transition-delay
    • transition-duration
    • transition-property
    • transition-timing-function [ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier(n, n, n)]

    Syntax

    div {
     transition: {property} {duration} {timing-function} {delay};
    }

  • Transform

    By modifying the coordinate space, CSS transforms change the shape and position of the affected content without disrupting the normal document flow.

    Properties

    • translate
    • scale
    • rotate
    • skew
    • perspective

    Syntax

    div {
     transform: translate(x,y);
     transform: scale(x,y);
     transform: rotate(angle);
     transform: skew(x-angle,y-angle);
    }

    Codepen

    Link

    Example

    Translate
    transform: translate(10px, -20px);
    Scale
    transform: scale(1.2);
    Rotate
    transform: rotate(90deg);
    Skew
    transform: skew(30deg,30deg);
    Transform Origin : right top
    transform: rotate(90deg);
    transform-origin: right top;

  • Animation

    Animations consist of two components, a style describing the CSS animation and a set of keyframes that indicate the start and end states of the animation's style, sa well as possible intermediate points

    Properties

    • @keyframes
    • animation-name
    • animation-duration
    • animation-delay
    • animation-iteration-count
    • animation-direction
    • animation-timing-function
    • animation-fill-mode
    • animation

    Syntax

    @keyframes example {
     from {width: 0%;}
     to {width:100%;}
    }     @keyframes example {
     0% {widht: 0%;}
     25% {widht: 25%;}
     50% {widht: 50%;}
     100% {widht: 100%;}
    }     animation: [name] [duration] [timing-function] [delay] [iteration-count] [direction] [fill-mode]; animation-name: example
    animation-duration: 5s
    animation-delay: 2s
    animation-iteration-count: 3,infinite
    animation-direction: normal, reverse,alternate,alternate-reverse;
    animation-timing-function:ease, linear, ease-in,ease-out,ease-in-out,cubic-bezier(n,n,n,n)
    animation-fill-mode: none, forwards, backwards, both

  • Back face visibility

    The backface-visibility property defines whether or not the back face of an element should be visible when facing the user.

    Syntax

    backface-visibility: visible|hidden|initial|inherit;

    Example

    Back face visibility
    transform: rotate3d(0, 1, 0, 180deg);
    backface-visibility: visible;

    Back face visibility
    transform: rotate3d(0, 1, 0, 180deg);
    backface-visibility: hidden;
    .object{
    perspective: 500px;
    transform-style: preserve-3d;
    }

    .object-child{
    height: 20px;
    width: 300px;
    background-color: red;
    transform: rotateY(20deg);
    position: absolute;
    }
